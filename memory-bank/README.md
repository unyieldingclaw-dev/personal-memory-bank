# Memory Bank - This Project's Context

This directory contains the Memory Bank for the **Memory Bank Standard** project itself.

Yes, we use our own standard to document our own standard. It's Memory Banks all the way down.

## Files

| File | Purpose |
|------|---------|
| `projectbrief.md` | What this standard is and its constraints |
| `systemPatterns.md` | Design decisions (template patterns, file structure) |
| `techContext.md` | Repository structure, file formats, IDE support |
| `activeContext.md` | Current focus and next steps |
| `progress.md` | What's complete and what's planned |

## Quick Commands

```powershell
# Check Memory Bank health
.\scripts\mb.ps1 status

# View this context
Get-ChildItem .\memory-bank\
```

## Meta Note

This Memory Bank demonstrates the standard in action:
- `projectbrief.md` shows a completed project brief
- `systemPatterns.md` shows architectural decisions
- `techContext.md` shows repository documentation
- `activeContext.md` shows session tracking
- `progress.md` shows milestone tracking

Use these as additional examples when creating your own Memory Bank.
